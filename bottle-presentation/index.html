    Bottle is so cool

  • Single file Python module, it's very small
  • No Python dependencies
  • Easy Integration with third party template engines
  • Easily enables clean URI routes
  • Supports RESTful out of the box
  • Useful selection of plugins
  • Full support of Python 2 and Python 3

What does it have ?

  • Routing
  • Small abtraction over wsgi
  • Templates
  • Utilities
  • In built Httpserver

What it does NOT have ?

  • In built Session support
  • In built form serialization and validation system
  • A caching framework , Unit Testing framework
  • An internationalization system

Where would I use it

  • Want to have lots of flexibility
  • Build your own larger framework
  • Just build something really simple
  • Rapid prototyping
  • Perfect learning tool for those just getting starting with web development

STATIC FILES 

          
@app.route('/static/<filename:re:.*\.jpeg>')

def send_image(filename):

    return static_file(filename, root='./static')



@app.route('/download/<filename:path>')

def down_image(filename):

    return static_file(filename, root='./static', download=filename)
